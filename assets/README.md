# Aseyori4U – Frank The Goat

Frank the Goat is an illustrated children’s story created under the Aseyori4U
“Success For You – Step-Up” learning project.

## Project Includes
- Full illustrated children’s book
- Print-ready PDF edition
- EPUB digital edition
- Original artwork files

## Purpose
To inspire young readers with positive life lessons about safety,
friendship, and making wise choices.
